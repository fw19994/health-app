日期选择器组件演示

组件介绍

日期选择器是一个可复用的组件，可以嵌入到任何需要日期选择功能的页面中。支持年月日选择，并提供多种配置选项。

示例 1：基本用法

已选日期：未选择
// HTML: <div id="datePickerContainer1"></div> // JavaScript: const datePicker1 = new DatePicker({ containerId: 'datePickerContainer1', onDateSelect: (date) => { document.getElementById('selectedDate1').textContent = `${date.getFullYear()}年${date.getMonth() + 1}月${date.getDate()}日`; datePicker1.hide(); } }); document.getElementById('showDatePicker1').addEventListener('click', () => { datePicker1.show(); });

示例 2：自定义配置

已选日期：未选择
// HTML: <div id="datePickerContainer2"></div> // JavaScript: const datePicker2 = new DatePicker({ containerId: 'datePickerContainer2', initialDate: new Date(2025, 5, 15), // 2025年6月15日 startYear: 2024, endYear: 2030, onDateSelect: (date) => { document.getElementById('selectedDate2').textContent = `${date.getFullYear()}年${date.getMonth() + 1}月${date.getDate()}日`; datePicker2.hide(); } }); document.getElementById('showDatePicker2').addEventListener('click', () => { datePicker2.show(); });

API 文档

构造函数选项

  • containerId: 容器元素ID，默认为 'datePicker'
  • onDateSelect: 日期选择回调函数
  • initialDate: 初始日期，默认为当前日期
  • startYear: 起始年份，默认为当前年份减5年
  • endYear: 结束年份，默认为当前年份加5年

方法

  • show(): 显示日期选择器
  • hide(): 隐藏日期选择器
  • getDate(): 获取选中的日期
  • setDate(date): 设置日期
